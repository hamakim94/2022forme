### 최근 시간 뽑기

~~~ sql
SELECT writername, createtime as '날짜'  
FROM board 
WHERE (writername , createtime )
in ( select writername, max(createtime)
	FROM board
    WHERE writername = '토르'
    group by writername
    )
;
~~~

- 조건이 2개인 경우에는 ,()로 묶어서 in 사용 가능함.
